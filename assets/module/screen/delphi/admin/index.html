Delphi Admin Tools

{{ trans.REPORT | trn }} Description
  Expiring Demos Report This utility can generate a report of all demo devices expiring within a user-specified period of time. By default the report will generate a report of all devices expiring in 30 days.
  Add Devices This utility allows Delphi Admin to add new Devices to ATT's System.
  Missing Maintenance Schedules This utility generates a report of vehicles without maintenance schedules.
  Time From Born to OTA Complete This reporting utility allows you to compare the time a device was "Born" with the time that its OTA Completed. The report you generate includes all devices that we've received Born Alerts for, along with the time of the 'OTA Complete' alert indicating that software has been successfully updated.
  Admin Support Tools This page contains a number of additional support tools available to vehimatics admin.
