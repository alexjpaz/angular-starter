FAQ Example



<div common-help tagged="maintenance-center, !tooltip, !popover" anytags="true" faq-title="FAQ: Maintenance Center"></div>

  • Add the "common-help" attribute to any HTML element to replace it with the above "Need Help" FAQ.
  • To specify topics, supply tags as a comma-separated string to the "tagged" attribute. To exclude a tag, prefix it with an exclamation mark.
  • By default, tags are queried with AND functionality, meaning it will only return topics that match ALL supplied tags. To query with OR functionality and return all topics that match any supplied tags, add an attribute of anytags="true"
  • To specify an FAQ title, use the faq-title attribute. If a title isn't specified, the title will default to "FAQ".
  • NOTE: the FAQ Data Model has been expanded so that each topic now has a "formats" object. Each topic will need different definitions for tooltip, popover, and FAQ implementations, and the "formats" object stores them.

    When adding or editing a topic, you will see an "Add Format" button and dropdown. Simply select the format you want to add, click "Add Format". An input and content preview window will be added to the page so you can add shorter definitions for tooltips or popovers. The directives will look for their specific format first, but default to using the main FAQ content if the directive format isn't available.
