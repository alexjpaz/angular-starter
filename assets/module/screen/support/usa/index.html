{{ trans.REPORT | trn }} Description
  User Lookup & Password Reset This utility allows you to lookup a user by login id, email, phone, or last name. You can verify their identity using their security questions, and if necessary reset their current password and have a temporary password sent to their primary email address.
  Receiving Delayed Alerts This utility analyzes the various communication nodes involved in sending an alert to help diagnose why a user might be receiving delayed alerts and attempts to offer possible causes and solutions.
  Not Receiving Alerts This utility runs a series of checks to help diagnose why a user isn't receiving an alert (or alerts) and, if available, provides a recommended action & url to resolve the issue.
  Troubleshooting: Device Lights This utility asks which device lights are on, off, or blinking to attempt to diagnose if the device is functioning as expected and provide recommendations to improve device communication or functionality.
  Troubleshooting Device This utility asks a number of questions to troubleshoot device issues that can prevent normal functioning including connection and communication problems. It includes light checking v1 and asks additional questions about whether the device has a clear line of sight. This version is in the help section in production.
  DTC Check This utility can be used to check if Delphi's DTC server is experiencing problems or not. Enter a valid DTC to see it's name and description. If you enter code P0301 and do not get results, Delphi's DTC Translation Service may be down. This utility can also be used to look up the raw descriptions of various DTC's.
